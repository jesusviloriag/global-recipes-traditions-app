# global-recipes-traditions-app
A React Native App for making and sharing global recipes with GPS and text recognition

Run in Android
```
npx react-native run-android
```

Run in iOS
```
npx react-native run-ios
```

Run Backend
```
cd backend
./mvnw
npm start
```
